# Ivy-Hive

A cottagecore-themed schedule and goal-tracking web app.

## Landing page

`index.html` is a self-contained landing page for Ivy-Hive — no build step required.
Open it directly in a browser, or enable **GitHub Pages** (Settings → Pages → Deploy from branch → `main` / root) to serve it live.

**Design system**
- Colors: beige `#F5EFE3` (base), sage `#A8B79A` (secondary), forest green `#3B5D42` (primary), honey-amber `#D9A24B` (accent)
- Type: Fraunces (headings), Nunito (body)
- Signature elements: hand-drawn ivy vine dividers, a honeycomb-clock hero illustration, and a hexagon-shaped "Get Started" button
